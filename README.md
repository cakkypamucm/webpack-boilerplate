This is `webpack` boilerplate (`vue`, `ts`, `scss`, `postcss`, `browserslist`, `babel`, `eslint`, `stylelint`, `prettier`) with [`nuxt`-like](https://nuxt.com/docs/guide/directory-structure) directory structure.

### Lint

```shell
npm run lint
```

### Develop

```shell
npm start
```

### Build

```shell
npm run build
```

### Docker

```shell
docker compose up
```
